# Goró da Mansão: Digital Experience

Este projeto foi importado do **Stitch** para o Antigravity.

## Estrutura do Projeto
- `index.html`: Home Page do projeto.
- `catalog.html`: Catálogo de produtos.
- `checkout.html`: Fluxo de pagamento.
- `about.html`: Informações sobre a marca.

## Tecnologias Utilizadas (no Stitch)
- HTML5 / CSS3
- Tailwind CSS (via CDN)
- Google Fonts (Space Grotesk, Manrope)
- Material Symbols Outlined

## Notas de Importação
Os arquivos foram baixados diretamente do servidor do Stitch em 25 de abril de 2026.
As imagens e recursos externos são carregados via URLs do Google/Stitch.
